This repository contains all materials, code, and resources for a semester-long independent study focused on **acquisition, visualization, and forecasting of financial time series data**.  

Students will learn how to retrieve real-world stock market data, explore and preprocess it, visualize trends, and build statistical and machine-learning models to forecast financial time series.

---

## Course Objectives

By the end of this study, students will be able to:

- Access stock market data using public APIs  
- Clean, preprocess, and structure time-series data  
- Visualize price movements, volatility, and trends  
- Apply statistical and machine learning models for forecasting  
- Evaluate model performance and interpret results  
- Produce reproducible, research-quality analysis  

---

## Repository Structure

All course materials in this repository are organized **by week of lecture** to support a structured, progressive learning path.

Each week may include:
- Lecture notes or slides  
- Jupyter notebooks  
- Code examples  
- Assignments or exercises
